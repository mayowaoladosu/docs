## Networking Infrastructure

Nubis provides a robust set of networking features to help you build secure and scalable environments for your applications.

### Virtual Private Clouds (VPC)

VPCs allow you to create isolated private networks for your resources. Instances within the same VPC can communicate over high-speed private links without exposing traffic to the public internet.

- **Isolation**: Each VPC is completely isolated from other networks.
- **Custom Subnets**: Define your own IP address ranges.
- **Internal Communication**: Low-latency communication between services.

### Load Balancers

Our managed Load Balancers distribute traffic across multiple instances to ensure high availability and performance.

- **Health Checks**: Automatically route traffic only to healthy instances.
- **SSL Termination**: Offload SSL/TLS encryption to the load balancer level.
- **Traffic Rules**: Define rules for routing traffic based on path, hostname, or port.

### Firewalls

Secure your infrastructure with our easy-to-use cloud firewalls. You can define rules to control inbound and outbound traffic at the network level.

- **Rule Management**: Create rules based on port, protocol (TCP/UDP/ICMP), and IP range.
- **Group Application**: Apply a single firewall to multiple instances simultaneously.
- **Dynamic Updates**: Changes take effect immediately across all associated resources.

### Domain Management

Nubis provides integrated DNS management to help you map your domain names to your cloud resources.

- **A/AAAA Records**: Map domains to IP addresses.
- **CNAME Records**: Create aliases for your services.
- **MX Records**: Configure your email servers easily.
