## Managed databases for application teams

Nubis Databases is the managed data-service layer for teams that want production-ready database operations without turning every workload into a database-administration project.

## What teams generally want from the service

- Supported engines matched to real application workloads
- Isolated network placement inside private environments
- Backup and recovery paths that are easier to operate
- Access control that fits the organization and project model
- Monitoring and scaling visibility tied to the rest of the platform

## How Nubis Databases fits into the platform

- Pair it with [Nubis Compute](/services/nubis-compute) when applications run on virtual machines.
- Keep it near the right [VPC design](/services/nubis-compute/networks-security/vpcs).
- Review [Billing and payments](/get-started/billing-payments) before production adoption.
- Use [Need help?](/need-help) when you need troubleshooting or rollout guidance.
